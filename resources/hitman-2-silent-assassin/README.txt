Manual Instructions (English)

1) Ultrawide screen

By default, dgVoodoo will use the "desktop" resolution, you should not change this.
But, a resolution is also defined in the h2.ini file, with default values Width = 1920, Height = 1080

To edit this file : 
- Go to the game directory, then scripts
- Open h2.ini
- Edit Width and Height values (set your screen resolution, for example Width = 3440, Height = 1440)
- Save the file

2) UI too little 

If UI is too little, you can set lower values in the previous file, while keeping the ratio of your screen.

For examples : 
- for a 16/9 2560x1440 screen, you can set Width = 1920, Height = 1080
- for a 16/9 3840x2160 screen, you can set Width = 2560, Height = 1440
- for a 21/9 3440x1440 screen, you can set Width = 2560, Height = 1080
- for a 32/9 5120x1440 screen, you can set Width = 3840, Height = 1080

3) Controller Patch 

Be careful with it; it will disable keyboard controls.

To enable/disable the controller patch:
- Go to the game directory
- Open Hitman2.ini
- Look at the bottom for the "=== CONTROLLER SUPPORT ===" area
- Comment out (prefix with ;) or uncomment (remove ;) these two lines
- Save the file

Instructions manuelles (French)

1) Écran ultrawide

Par défaut, dgVoodoo utilise la résolution "bureau" et c'est bien ainsi, ne modifiez pas.
Cependant, une résolution est également définie dans le fichier h2.ini, avec comme valeurs par défaut : Width = 1920, Height = 1080.

Pour modifier ce fichier :
- Allez dans le répertoire du jeu, puis dans « scripts ».
- Ouvrez h2.ini.
- Modifiez les valeurs de largeur et de hauteur (définissez la résolution de votre écran, par exemple : Width = 3440, Height = 1440).
- Enregistrez le fichier.

2) Interface trop petite

Si l'interface est trop petite, vous pouvez définir des valeurs inférieures dans le fichier précédent, en respectant les proportions de votre écran.

Par exemple :
- Pour un écran 16/9 de 2560 x 1440, vous pouvez définir Width = 1920, Height = 1080.
- Pour un écran 16/9 de 3840 x 2160, vous pouvez définir Width = 2560, Height = 1440.
- Pour un écran 21/9 de 3440 x 1440, vous pouvez définir Width = 2560, Height = 1080.
- Pour un écran 32/9 de 5120 x 1440, vous pouvez définir Width = 3840, Height = 1080.

3) Patch Manette 

Attention avec le patch manette, il desactivera les commandes au clavier.

Pour activer/desactiver le patch manette : 
- allez dans le répertoire du jeu
- ouvrez Hitman2.ini
- chercher en bas la zone "=== CONTROLLER SUPPORT ===" 
- commentez (en préfixant par ;) ou décommentez (en enlevant ;) ces 2 lignes
- sauvegardez le fichier
